Линейный поиск — один из самых простых алгоритмов поиска.
Здесь нужно последовательно проверять каждый элемент списка,
начиная с первого, пока не будет найден искомый элемент или
не будут проверены все элементы. Таким образом алгоритм 
гарантированно находит элемент, если он присутствует в массиве.

Сложность по времени
Худшее время O(n)
Среднее время O(n)
Лучшее время O(1)

Затраты памяти O(1)
