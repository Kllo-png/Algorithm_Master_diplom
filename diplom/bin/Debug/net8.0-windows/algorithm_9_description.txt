Прыжковый поиск — это алгоритм поиска в отсортированном массиве, который
сочетает идеи линейного и бинарного поиска. Здесь нужно прыгать фиксированными
шагами по массиву до тех пор, пока не будет найден блок, содержащий искомый элемент,
после чего выполнить линейный поиск в этом блоке. Оптимальный размер шага обычно 
принимается равным √n.

Сложность по времени
Худшее время O(√n)
Среднее время O(√n)
Лучшее время O(1)

Затраты памяти O(1)
